# Advanced Numpy & Pandas

## Install

* optional: create conda environment
  * `conda env create -f pymotw_numpy.yml`
  * `conda activate pymotw_numpy`

## Topics

* Advanced Numpy
  * Anatomy of an array
  * Views & Copies
  * Broadcasting
* Pandas
  * Series & Dataframes
  * Multiindex
  * Groupby
